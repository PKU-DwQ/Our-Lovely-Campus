    世界从不缺乏新奇，缺的是凝视旧事物时，那一瞬恍若初见的停顿。

### 文件配置

    因为本程序涉及接入大模型, 且api不方便上传至git, 所以需要自行配置

#### 具体方法

仿照`config_demo.json`文件,创建`config.json`文件
内容如下
```json
{
    "api_key": "Bearer <your_api_key>" 
    //其中<your_api_key>需要替换成真实的api key
}
```
* 注意: `Bearer `是格式的一部分, 请不要自行删除
