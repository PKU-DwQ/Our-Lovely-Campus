### 文件配置

    因为本程序涉及接入大模型, 且api不方便上传至git, 所以需要自行配置

#### 具体方法

仿照`config_demo.json`文件 *(但请不要修改该文件)* , 创建`config.json`文件
写入内容:
```json
{
    "_instructions": "替换<your_api_key>为真实密钥",
    "api_key": "Bearer <your_api_key>"
}
```
* 注意: `Bearer `是格式的一部分, 请不要自行删除
